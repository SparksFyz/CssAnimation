# CssAnimation
This is a repository for learning css animation

## how to use

### Install gulp

* `npm install --global gulp`
* create `gulpfile.js`
* run `gulp`


## Introduction

## Demo List

### Navbar

> Fade in/out when scrolling down/up.

[Demo](http://sparksfyz.github.io/CssAnimation/navbar/navbar.html)
